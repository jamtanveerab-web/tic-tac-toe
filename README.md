# ❌⭕ Tic Tac Toe Game

A classic two-player Tic Tac Toe game built with HTML, CSS, and JavaScript. Play with a friend and see who wins!

## 🎮 How to Play

- Player 1 uses **X**
- Player 2 uses **O**
- Click on any empty box to place your mark
- First to get 3 marks in a row (horizontal, vertical, or diagonal) wins
- If all 9 boxes are filled with no winner, it's a tie

## ✨ Features

- Clean and minimal design
- Two-player gameplay
- Automatic win detection
- Draw/Tie detection
- Reset button to start new game
- Displays winner message
- Responsive design (works on mobile)

## 🛠️ Technologies Used

- HTML5
- CSS3 (Grid layout)
- JavaScript (ES6)
